01 · AI 모델 구조 설계

최신 모델을 시도했지만, 근거를 갖고 기각했다

기본 선택은 대회 제공 YOLOv7 가중치(mAP@0.5 98.1%, 약 20만 장 학습). 대안 아키텍처를 실제로 학습·검증한 뒤 데이터 격차를 근거로 유지 결정.

시도학습 조건결과판단
YOLO11m 재학습 18,392장, 50 epoch mAP 98.4% (유사) · order F1 73.4% (−23.2%p) 기각
YOLO11m + Knowledge Distillation 동일 데이터 order F1 91.1% (KD로 개선, YOLOv7 대비 −7.5%p) 기각
YOLOv7 자체 파인튜닝 Cut&Paste 5,000장 + 30 epoch mAP@0.5 0.9904 → 0.9840 (하락) 기각
대회 제공 YOLOv7 가중치 약 20만 장 사전학습 mAP@0.5 98.1% · 파이프라인 전체 최고 성능 채택
핵심 판단 근거 — 대회 가중치는 20만 장, 자체 학습은 1.8만 장으로 10배 이상 데이터 격차. mAP는 비슷해도 실제 이벤트 검출력에서 크게 밀림 — "최신 모델 ≠ 정답", 데이터 규모가 확보된 모델을 쓰는 게 이 과제에서 더 합리적.

03 · Multi-Camera 활용 전략

획일적 규칙이 아닌, 클래스별 4단 제어 레이어

L4동적 카메라 가중치클래스 자신의 confidence로 실시간 occlusion 판단, 투표권 조절
